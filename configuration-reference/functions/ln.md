# ln

Natural log of value i.e. ln(col1)

## Signature

> LN(col1)

## Usage Examples


```sql
select ln(1) AS value
from ignoreMe
```

| value |
| ------------- | 
| 0 |


```sql
select ln(12) AS value
from ignoreMe
```

| value |
| ------------- | 
| 2.4849066497880004 |
