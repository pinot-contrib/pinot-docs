# CEIL

Rounded up to the nearest integer.

## Signature

> CEIL(col1)

## Usage Examples


```sql
select CEIL(12.1) AS value
from ignoreMe
```

| value |
| ------------- | 
| 13|


```sql
select CEIL(-12.1) AS value
from ignoreMe
```

| value |
| ------------- | 
| -12 |
