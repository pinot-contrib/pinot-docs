# count

Get the count of rows in a group

## Signature

> COUNT(colName)

## Usage Examples

These examples are based on the [Batch Quick Start](../../basics/getting-started/quick-start.md#batch).

```sql
select count(*) AS value
from baseballStats 
```

| value   | 
| ------------- |
| 97889 |
